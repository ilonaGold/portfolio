Async Race Game

Read more

Live site Code repo

Async Race is a dynamic single-page racing game built with TypeScript, offering car management, real-time race animations, and a feature-rich Garage for creating, editing, and deleting cars with custom colors. The Winners page tracks detailed race statistics, and all operations are handled via a RESTful API. To address cold starts on free hosting, I implemented a keep-alive system with GitHub Actions that regularly pings both backend and frontend. The solution demonstrates practical DevOps skills in addressing real-world hosting limitations while maintaining cost efficiency. A custom animated loader ensures seamless, engaging interactions at every step.
